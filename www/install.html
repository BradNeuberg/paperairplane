Install Paper Airplane

You must have the following already installed to use Paper Airplane at this time:

  • JRE 1.5+ must be installed in the browser. Download it here.
  • Only Windows is supported at this time.
  • FireFox/Mozilla Only. Internet Explorer is not supported.

Warning: This is alpha-level software, so please only install it in non-critical installations of FireFox and Mozilla. Also, there is no uninstall function yet, but FireFox makes it possible to disable extensions.

Click here to auto-install Paper Airplane 0.1.2

If you are using FireFox 1.0, then you will have to click the 'Edit Options' button that will appear in your browser after clicking this link, and then press ALLOW on the dialog that comes up. By default FireFox 1.0 only allows extensions to be installed from update.mozilla.org. When Paper Airplane is ready for a final, stable release I will move it there; until now while it is in development installation should be done through this page.
