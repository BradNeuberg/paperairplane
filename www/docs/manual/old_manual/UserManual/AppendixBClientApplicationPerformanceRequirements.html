Appendix B - Client Application Performance Requirements

The only exact metric for client application performance is startup time. Starting the application the first time (without the Quick Starter in memory) should take no more than 9 seconds. When the Quick Starter is enabled and the application is partially in memory, startup should take one and a half seconds.

The rest of the application should exhibit reasonable speed. In particular, nothing should feel 'sluggish' (independent of network access).
