Main Portions of Paper Airplane

The Paper Airplane application has two modes: Viewing Mode and Editing Mode. Viewing Mode occurs during normal browsing of a web site. Editing Mode is invoked when a user is editing or creating a new web page.

There are also several other major portions of the program. The first is The Two Way Web Directory, which provides an easy, hierarchical way of finding Two Way Web Sites that match your interests; this directory is the default home page and is shown when Paper Airplane is first run. Paper Airplane also seamlessly integrates with Windows by adding a My Web Sites on the Two Way Web entry in the My Network Places folder on your desktop. Use this virtual folder to access the Two Way Web Sites that you are a member of, and to save and edit materials on your site using the Windows File Explorer and third-party applications. The final major portion of the program is the Paper Airplane icon in the Windows taskbar; even when you quit the browser this icon remains in your taskbar, providing quick-start functionality as well as continuing to serve and store content on the Two Way Web.
