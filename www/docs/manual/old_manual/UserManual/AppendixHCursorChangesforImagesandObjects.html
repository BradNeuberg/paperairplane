Appendix H - Cursor Changes for Images and Objects

The mouse cursor should behave the same when browsing both older World Wide Web HTML pages and Two Way Web pages. When the mouse is over text that is selectable, the cursor should change into a text caret. When the mouse is over a blank area of the page, an image, or an object, the mouse should change into the standard Windows arrow pointer. If the mouse is over a link it should change into a hand with one finger pointing. These are all standard Windows mouse cursors.
