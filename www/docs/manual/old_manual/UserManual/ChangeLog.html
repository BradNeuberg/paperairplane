Change Log

 

User Manual Version 1.1 (current version)

  1. Added this change log
  2. Changed how the domain ending system works from a user-standpoint.
    1. Reduced the number of domain endings, in order to simplyify the UI in the New Site Wizard when a user picks a domain ending. Chose about 140 of the best, most commonly used or creative endings.
    2. Got rid of the filtering buttons in the Choosing a Name step in the New Site Wizard.
    3. When a user makes a custom domain ending now, it does not show up in the list of domain endings for other users to choose when creating their own sites.
  3. Modified the introduction (Introduction.html) to have screenshots.
