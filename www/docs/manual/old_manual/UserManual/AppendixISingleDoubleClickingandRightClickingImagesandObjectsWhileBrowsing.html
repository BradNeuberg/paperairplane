Appendix I - Single and Double-Clicking Images, Objects, and Text While Browsing

Single, double-clicking, and right-clicking image and objects should behave the same when browsing both older World Wide Web HTML pages and Two Way Web pages. Single-clicking does nothing. Double-clicking an image or object also does nothing, but double-clicking text causes the word double-clicked to be selected.
