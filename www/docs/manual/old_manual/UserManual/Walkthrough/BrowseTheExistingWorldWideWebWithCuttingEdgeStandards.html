Walkthrough - Next-Generation Browsing

Browse the Existing World Wide Web with Cutting-Edge Standards


Paper Airplane is built to replace Internet Explorer, Mozilla, and Netscape. It is fully compatible with browsing the existing World Wide Web, so you can rest assured that you can still view your existing news, entertainment, and information sites using Paper Airplane. Paper Airplane also moves the web forward in important ways by supporting key, cutting-edge standards, such as:

  • HyperText Markup Language (HTML) version 4.0
  • Cascading Style Sheets (CSS) version 1 with partial support for version 2
  • Document Object Model (DOM) versions 1 and 2
  • eXtended Markup Language (XML) 1.0
  • JavaScript 1.5
  • Resource Description Format (RDF)
  • Math Markup Language (MathML)

Paper Airplane also takes traditional client-server applications to the next level. Create advanced, fast intranet and internet applications using Paper Airplane's advanced XML-based user-interface language, XML User-interface Language (XUL). XUL makes cross-platform user interfaces as easy to build and customize as web pages. Use Paper Airplane's robust JavaScript and web-services support to fully script your traditional server-side applications. If you run into problems during development, use the integrated JavaScript debugger or DOM Inspector to iron out issues.
