Appendix J - Menus that Appear When Right-Clicking

This section does not cover all right-click context menus; it simply covers ones that have not been covered previously in this document.

It is important that right-click menus work the same on both older World Wide Web HTML pages and while browsing Two Way Web Pages in order to provide user-interface consistency.

Right-clicking in blank areas shows the following options:

  • Back
  • Forward
  • Reload
  • Stop
  • -------------------------
  • Bookmark This Page
  • Save Page As...
  • Send Page...
  • -------------------------
  • Edit Page
  • -------------------------
  • View Background Image
  • -------------------------
  • View Page Source (disabled for Two Way Web Pages)
  • View Page Info (shows the About this Page... dialog for Two Way Web Pages)

 

Right-clicking on images shows the following options:

  • View Image
  • Block Images from this Server (Disabled for Two Way Web Pages)
  • Copy Image Location (Disabled for Two Way Web Pages)
  • --------------------------
  • Save Image As...
  • Set As Wallpaper (Disabled for Two Way Web Pages)
  • --------------------------
  • Properties

 

Right-clicking on selected text shows the following options:

  • Undo
  • --------------------------
  • Cut
  • Copy
  • Paste
  • Delete
  • --------------------------
  • Select All
  • --------------------------
  • Web Search for "selected text"

 

Right-clicking on links shows the following options:

  • Open Link in New Window
  • Open Link in New Tab
  • Edit Link in New Window
  • ---------------------------
  • Bookmark This Link
  • Save Link Target As...
  • Copy Link Location
  • ---------------------------
  • Properties

 

Right-clicking on embedded spreadsheets in Two Way Web pages currently shows nothing.
